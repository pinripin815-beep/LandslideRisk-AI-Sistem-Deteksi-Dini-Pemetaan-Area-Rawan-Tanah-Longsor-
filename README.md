# LandslideRisk AI

Proyek sederhana berbasis Python untuk mendeteksi area rawan tanah longsor menggunakan Machine Learning (**Random Forest Classifier**).

## 📊 Cara Kerja Sistem
Program ini mensimulasikan data lingkungan yang terdiri dari:
1. **Kemiringan Lereng** (dalam derajat)
2. **Intensitas Curah Hujan** (mm/bulan)
3. **Kepadatan Vegetasi** (indeks NDVI)

AI akan mempelajari pola dari data tersebut untuk menentukan apakah suatu wilayah masuk dalam kategori **Aman** atau **Rawan Longsor**.

## 🛠️ Fitur Kode
- Pembuatan data simulasi lingkungan otomatis.
- Pembagian data untuk pelatihan (training) dan pengujian (testing).
- Prediksi otomatis untuk data area baru yang dimasukkan.
